# Excel Data Analytics Practice

This repository contains my Excel practice work completed as part of my Data Analytics learning journey. The purpose of this repository is to build a strong foundation in Excel by working with datasets and applying commonly used analytical techniques.

**Topics**

**Excel Formulas**
- SUM()
- AVERAGE()
- COUNT()
- IF()
- COUNTIF()
- SUMIF()
- VLOOKUP()
- XLOOKUP()

**Conditional Formatting**
- Highlighting duplicate values
- Identifying missing values
- Data bars and color scales
- Rule-based formatting

**Charts and Visualization**
- Column Charts
- Bar Charts
- Line Charts
- Pie Charts

**Files Included**

- Formulas.xlsx
- Conditional Formatting.xlsm
- Charts.xlsm

**Tools Used**

- Microsoft Excel

**Learning Outcome**

Through this practice, I improved my understanding of Excel functions, data formatting, and data visualization techniques. This repository represents my hands-on learning process while developing the Excel skills required for Data Analytics.
This repository is created for learning and practice purposes as I continue to strengthen my Data Analytics knowledge and prepare for real-world data analysis projects.
